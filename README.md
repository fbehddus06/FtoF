# FVG 스캐너 · 폰 전용 셋업 가이드

매 4시간마다 자동으로 시장을 스캔해서, 조건 맞는 종목이 발견되면
**텔레그램으로 차트와 매매 정보를 보내주는** 봇입니다.

서버 비용 없음 · 코드 수정도 폰에서 가능.

---

## 작동 방식

```
GitHub Actions (4시간마다 자동 실행)
        ↓
fvg_scanner.py 가 Bybit에서 상위 40개 종목 데이터 수집
        ↓
각 종목에 대해: FVG·피보·LVN 조건 검사
        ↓
조건 만족 종목 → 차트 그려서 텔레그램으로 전송
        ↓
폰에서 알림 받음 → 거래소 앱으로 진입
```

---

## 1단계 · 텔레그램 봇 만들기 (3분)

폰의 텔레그램 앱에서:

1. 검색창에 `@BotFather` 입력 후 대화 시작
2. `/newbot` 입력
3. 봇 이름 입력 (예: `내FVG스캐너`)
4. 봇 ID 입력 (예: `my_fvg_scanner_bot` — 끝이 `bot`으로 끝나야 함)
5. **토큰을 복사** (예: `7891234567:ABCdefGHI...`)
6. 방금 만든 봇을 검색해서 `/start` 누르기 (안 누르면 메시지 못 받음)

**챗 ID 얻기:**
- 폰 브라우저에서 아래 주소 열기 (`<토큰>` 자리에 위에서 받은 토큰 붙여넣기):
  ```
  https://api.telegram.org/bot<토큰>/getUpdates
  ```
- 응답에서 `"chat":{"id":123456789` 부분의 숫자가 **챗 ID** 입니다.

---

## 2단계 · GitHub 저장소 만들기 (5분)

폰의 GitHub 앱(또는 모바일 브라우저 github.com)에서:

1. **새 저장소 생성** — 이름 자유 (예: `fvg-scanner`), Private으로 설정
2. 저장소에 파일 3개 업로드:
   - `fvg_scanner.py`
   - `requirements.txt`
   - `.github/workflows/scan.yml` (위 폴더 구조 그대로)

> 💡 폰에서 파일 업로드가 번거로우면 PC에서 한 번만 올리면 끝입니다.

---

## 3단계 · 시크릿 설정 (2분)

GitHub 저장소에서:

1. 저장소 페이지 → `Settings` 메뉴 (모바일에선 메뉴 아이콘 안에 있음)
2. 좌측 또는 하단의 `Secrets and variables` → `Actions`
3. `New repository secret` 클릭
4. 2개의 시크릿 추가:

| 이름                | 값                          |
|---------------------|----------------------------|
| `TELEGRAM_TOKEN`    | 1단계에서 받은 봇 토큰     |
| `TELEGRAM_CHAT_ID`  | 1단계에서 받은 챗 ID       |

---

## 4단계 · 테스트 실행 (1분)

자동 실행을 기다리지 않고 지금 바로 테스트:

1. 저장소 → `Actions` 탭
2. 좌측 `FVG Scanner` 워크플로 선택
3. 우측 `Run workflow` 버튼 클릭
4. 1~2분 후 텔레그램에 알림 도착 확인 ✓

---

## 설정 변경

`scan.yml` 파일에서 환경변수만 바꾸면 됩니다:

```yaml
INTERVAL: '4h'        # 봉 단위: 15m, 30m, 1h, 2h, 4h, 6h, 12h, 1d
TOP_N: '40'           # 스캔할 종목 수 (10 ~ 100)
MIN_SCORE: '30'       # 알림 최소 점수 (낮출수록 알림 많아짐)
EXCHANGE: 'bybit'     # bybit, binance, okx 중 선택
MAX_ALERTS: '5'       # 한 번에 보낼 최대 알림 수
```

스캔 주기 변경: `cron: '5 */4 * * *'` 부분 수정
- `'5 */1 * * *'` — 매 1시간마다
- `'5 */2 * * *'` — 매 2시간마다
- `'0 9 * * *'` — 매일 한국시간 18:00 (UTC 9시)

## 자주 묻는 질문

**Q. GitHub Actions 정말 무료인가요?**
A. Private 저장소는 월 2000분 무료. 한 번 스캔에 1~2분이니까 매 4시간 실행해도 월 ~360분이면 충분합니다.

**Q. 자동매매까지 연결할 수 있나요?**
A. 가능합니다. CCXT가 이미 들어있으니 거래소 API 키를 시크릿에 추가하고, `format_caption` 다음에 진입 주문 코드를 넣으면 됩니다. 다만 신호의 신뢰도를 충분히 검증한 후에 하세요.

**Q. 시그널이 너무 자주 와요 / 너무 안 와요.**
A. `MIN_SCORE` 값 조정. 높이면 적게, 낮추면 많이.

**Q. 같은 종목 중복 알림이 옵니다.**
A. 4시간마다 같은 셋업이 그대로 유효하면 다시 알림이 갑니다. 중복을 막으려면 마지막 알림 시점을 저장하는 로직 추가 필요 (요청하시면 만들어드림).

**Q. 다른 전략 조건도 추가하고 싶어요.**
A. `analyze()` 함수가 진입 조건의 핵심입니다. 거기에 RSI, MACD 등 추가 필터를 넣으면 됩니다.
